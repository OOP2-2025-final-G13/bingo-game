## 概要

このアプリケーションは、インストール不要でブラウザからすぐに使えるビンゴ抽選システムです。
PCやスマートフォンなど、デバイスを問わずに利用でき、プロジェクターに投影してイベントで使用することも想定しています。
抽選だけでなく、参加者用のビンゴカード生成機能や、最短ビンゴ記録の自動集計機能も備えています。

## アピールポイント

1. ストレスフリーな操作性とレスポンシブデザイン:
    *  1画面完結: 抽選とカード管理（最大2枚）を同一画面で操作可能。司会者がプレイヤーを兼ねる少人数プレイに最適です。
    *  完全レスポンシブ: PCの大画面からスマホまで、常に最適なレイアウトで表示されます。
    *  統一されたデザイン: 全画面でデザインを統一し、直感的な操作感を実現しました。

2. 堅牢なデータ管理と不正防止機能:
    *  データベース連携: Flask + SQLiteにより、ブラウザを閉じても「最短記録」や「履歴」が消えない設計です。
    *  不正防止ロジック: 実際に抽選された数字しかマークできない仕組みを実装し、公平なゲームを保証します。

3. 「見やすさ」へのこだわり:
    *  履歴の可視化: 履歴画面では最新の数字を強調表示。途中参加者も状況がすぐに分かります。
    *  最短記録の自動集計: 面倒な記録管理もシステムにお任せ。常にベストスコアがトップに表示されます。

## 主な機能

* ビンゴ抽選機能: 1〜75の数字をランダムに抽選し、大きく表示します。
* ビンゴカード機能: プレイヤー用の5x5ビンゴカードを生成・表示・操作できます。
* 履歴表示機能: 抽選された過去の数字一覧をグリッド形式で確認できます。
* 最短記録管理: 何手でビンゴしたかの「最短記録」をサーバーに自動保存し、常にトップ画面に表示します。
* リセット機能: データベースの記録も含め、全てを初期状態に戻すことができます。

## セットアップ

### 必要パッケージ

本アプリケーションは Python の Flask フレームワークを使用しています。

```bash
# 依存ライブラリのインストール
pip install -r requirements.txt
```

### 起動方法

以下のコマンドを実行してサーバーを立ち上げてください。

```bash
python app.py
```

## 使い方

1. 起動: ブラウザで `http://127.0.0.1:5500` にアクセスします。
2. カード生成: 「＋」ボタンを押してビンゴカードを生成します。
3. 抽選: 「抽選」ボタンを押して数字を引きます。
4. 確認: 画面中央に出た数字が表示されます。「履歴」ボタンで一覧を確認できます。
5. 記録: ビンゴが成立すると自動的に「BINGO！」が表示され、最短記録であればデータベースに保存されます。

## 技術スタック

*  Frontend: HTML5, CSS3, JavaScript
*  Backend: Python 3, Flask
*  Database: SQLite

## ディレクトリ構成

```
.
├── app.py                 # バックエンドのエントリーポイント (Flask)
├── bingo_records.db       # SQLiteデータベースファイル (自動生成)
├── requirements.txt       # 依存ライブラリ一覧
├── static/                # 静的ファイル群
│   ├── css/               # スタイルシート (base.css, index.css, effects.css)
│   └── js/                # JavaScriptロジック
│       ├── main.js        # メインコントローラー (BE1)
│       ├── logic.js       # ゲームロジック (BE1)
│       ├── bingodata.js   # データ管理 (BE2)
│       ├── bingorecord.js # 通信・記録管理 (BE2)
│       └── effects.js     # エフェクト・演出 (FE2)
└── templates/             # HTMLテンプレート
    ├── index.html         # メイン画面 (FE1)
    └── history.html       # 履歴画面 (FE2)
```
